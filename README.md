📖 **詳細玩法說明**：[USAGE.md](USAGE.md)
🚀 **部署與優化指南**：[DEPLOY-AND-OPTIMIZE.md](DEPLOY-AND-OPTIMIZE.md)

---

## 快速開始

```bash
# 安裝
npm install

# 開發伺服器
npm run dev   # → http://localhost:3000

# 打包
npm run build
```

## 操作

| 鍵盤 | 動作 |
|---|---|
| ← / → | 切換車道 |
| ↑ / Space | 跳躍 |
| ↓ | 減速 |

**手機觸控**：長按左右切換車道並加速、輕點跳躍

**🥚 隱藏指令**：在開始畫面依序按 `↑↑↓↓←←→→AB` 啟動 God Mode
